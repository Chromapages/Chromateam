Agent Handoff Manager

API Reference & Endpoint Documentation

Base URL: http://localhost:3461/api
WebSocket: ws://localhost:3461 LIVE
System & Health
GET /
API root with endpoint list
GET /health
System health check
curl http://localhost:3461/health
Agents
GET /agents
List all agents with status and workload
GET /agents/:id/status
Get specific agent details and pending tasks
curl http://localhost:3461/api/agents/bender/status
Handoffs (Core)
GET /handoffs
List all handoffs sorted by newest first
POST /handoff
Create a new handoff between agents
curl -X POST http://localhost:3461/api/handoff \ -H "Content-Type: application/json" \ -d '{"fromAgent":"chroma","toAgent":"bender","task":"Build landing page","priority":"high"}'
POST /handoff/with-dependencies
Create handoff that waits for other handoffs to complete
curl -X POST http://localhost:3461/api/handoff/with-dependencies \ -d '{"fromAgent":"prism","toAgent":"pixel","task":"Write copy","dependsOn":["handoff_123"]}'
POST /handoff/:id/start
Mark handoff as in_progress
POST /handoff/:id/complete
Complete a handoff
POST /handoff/:id/cancel
Cancel a pending handoff
GET /context/:agentId
Get agent's pending handoffs and context
Templates & Pipelines
GET /templates
List all available templates
POST /templates
Create new template
POST /template/:name/execute
Execute template (async or sequential)
curl -X POST http://localhost:3461/api/template/full-pipeline/execute \ -d '{"task":"Build portfolio website","runAsync":true}'
POST /pipeline
Create custom agent pipeline
curl -X POST http://localhost:3461/api/pipeline \ -d '{"agents":["chroma","prism","pixel","bender"],"task":"Launch product"}'
GET /pipeline/:id
Get pipeline status and progress
POST /pipeline/:id/cancel
Cancel all pending steps in pipeline
Smart Routing
POST /auto-assign
Auto-route task based on keywords + workload
curl -X POST http://localhost:3461/api/auto-assign \ -d '{"task":"Fix login bug","priority":"urgent"}'
POST /task/:agentId
Send task directly to specific agent
POST /chat/:agentId
Send chat message to agent
Advanced Features
POST /parallel
Send task to multiple agents simultaneously
curl -X POST http://localhost:3461/api/parallel \ -d '{"fromAgent":"chroma","toAgents":["pixel","canvas"],"task":"Create assets"}'
POST /conditional
Conditional routing based on keywords
curl -X POST http://localhost:3461/api/conditional \ -d '{"trigger":"Build website","condition":{"type":"keyword","keywords":["build"]},"ifTrue":"bender","ifFalse":"pixel"}'
POST /escalate/:handoffId
Escalate stuck handoff up the chain
POST /schedule
Schedule handoff for future execution
curl -X POST http://localhost:3461/api/schedule \ -d '{"fromAgent":"chroma","toAgent":"bender","task":"Weekly report","scheduledAt":"2026-03-09T09:00:00Z"}'
POST /feedback/:handoffId
Submit rating/feedback for completed handoff
Context & History
GET /history
Query handoff history with filters
curl "http://localhost:3461/api/history?agent=bender&limit=10"
GET /similar
Find similar past tasks
curl "http://localhost:3461/api/similar?task=build landing page"
GET /autocomplete
Get task suggestion autocomplete
POST /context-aware
Get agent context suggestions
Dashboard & Metrics
GET /dashboard
Full dashboard with stats, agents, recent activity
curl http://localhost:3461/api/dashboard
GET /metrics
System metrics and analytics
File Upload
POST /upload
Upload files (multipart/form-data)
curl -X POST http://localhost:3461/api/upload -F "files=@design.pdf"
POST /handoff-with-files
Create handoff with file attachments
GET /handoff/:id/deliverables
Get files produced by a handoff/pipeline
Web Pages
GET /dashboard
Visual dashboard with live stats
